Why I built this.

  • This project was the third project in the responsive web design certification path at FreeCodeCamp.org.
  • I was challenged to build a product landing page with a nav bar, HTML5 validation, and full responsiveness using HTML, CSS, and Javascript.
  • The product I showcase on this landing page is a hobby of mine.I have aspirations to build an e-commerce web page to help promote, produce, and sell this product in the future.

What I learned.

  • I experimented with clip-path to give my webpage an asymmetrical design.
  • I built a mobile nav bar, also called a "hamburger" menu. Utilized some javascript to animate the mobile navigation.
  • Used HTML5s iframe element to embed a video onto my landing page.
