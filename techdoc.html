Why I built this.

  • This project was the fourth project in the responsive web design certification path at FreeCodeCamp.org.

What I learned.

  • This project focused on using media queries to create a responsive webpage.
  • The HTML5 "code" element can be used to display a text as if it was written in a text editor.
  • Sometimes teaching is the best way to learn.
